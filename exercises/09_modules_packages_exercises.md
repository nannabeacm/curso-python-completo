# Exercises: Modules (Gadgets)

## Mission 1: The Dice Roll (Random)
Create `dice.py`.
Import `random`.
Simulate rolling a 20-sided die (D20 for DnD fans).
Print a random number between 1 and 20.

## Mission 2: Time Travel (Datetime)
Create `timetravel.py`.
Import `datetime`.
Print the current date and time.
Calculate the date 100 days from now. (Hint: look up `timedelta`).

## Mission 3: Custom Gadget
1. Create a file `stark_tech.py`.
2. Define a function `repulsor_blast()` that prints "PEW PEW!".
3. Create a **new** file `battle.py`.
4. Import `stark_tech` inside `battle.py`.
5. Call `stark_tech.repulsor_blast()`.

## Mission 4: Check PIP
Open your terminal.
Run:
```bash
pip list
```
See what is already installed in your environment.
